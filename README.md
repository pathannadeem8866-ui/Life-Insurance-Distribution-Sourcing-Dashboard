# 🏢 Life Insurance Distribution & Sourcing Dashboard

## 📝 Overview
This project is an end-to-end data engineering and business intelligence solution designed to analyze the performance, recruitment ROI, and operational efficiency of an insurance agency network. Processing over 500,000 records, this dashboard provides actionable insights into agency management and sourcing pipelines.

## 🎯 Business Impact & Domain Focus
Leveraging deep domain expertise in sourcing and managing life insurance agents, this project tracks:
* **Recruitment ROI:** Evaluating the cost, ramp-up time, and success rate of newly sourced agents.
* **Performance Metrics:** Identifying top-performing distribution channels and agent productivity trends.
* **Operational Efficiency:** Streamlining the agency management lifecycle from initial onboarding to active sales.

## 🛠️ Tech Stack & Methodology
* **Data Processing:** Python (`Cleaning_data.py`)
* **Database Management:** SQL Server (`Insurance_SQL.sql`)
* **Data Visualization:** Power BI (`Insurance_Project.pbix`)
